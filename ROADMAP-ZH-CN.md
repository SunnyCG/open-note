# Open Note 开发路线图

本文档概述了 Open Note 跨平台桌面笔记应用的开发路线图。

## 当前版本：0.1.0

### 已完成功能

- [x] CodeMirror 6 Markdown 编辑器
- [x] Wiki 风格双向链接 (`[[笔记名称]]`)
- [x] 基于文件的仓库存储
- [x] 多仓库支持与侧边栏切换
- [x] 主题系统（深色/浅色 + 自定义主题）
- [x] 自动保存与状态指示
- [x] 文件夹组织
- [x] 文档大纲导航
- [x] 反向链接与出站链接面板

## 参与贡献

想要帮助塑造 Open Note 的未来？请查看我们的[贡献指南](CONTRIBUTING.md)和[开放问题](https://github.com/openagentapp/open-note/issues)。

### 优先贡献领域

1. **Markdown 功能** - 帮助实现完整的 GFM 支持
2. **标签系统** - 高需求的核心功能
3. **文档** - 用户指南和教程
4. **测试** - 单元测试和集成测试
